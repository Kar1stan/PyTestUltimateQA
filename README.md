### Git clone
This will clone the project,and install the Playwright+Pytest which is required to run the tet. No Selenium Server is required.
```
$ git clone https://github.com/Kar1stan/PyTestUltimateQA.git
$ cd PyTestUltimateQA
$ pypenv install
```
